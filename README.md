# Court Manager

A tennis doubles team management app built with HTML/CSS/JS (frontend) 
and Java Spring Boot (backend).

## Live Demo
https://andrewl746.github.io/CourtManager

## Note
The backend is hosted on Railway's free tier which spins down after 
inactivity. When you first open the site, wait about 30-60 seconds 
before adding players to give the backend time to wake up. Data is 
stored in memory so it resets when the server restarts, meaning players 
and lineups should be added and generated within the same session.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Java Spring Boot
- Algorithm: Custom pairing algorithm with merge sort
